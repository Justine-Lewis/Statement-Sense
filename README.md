# Statement-Sense
Web Application that accepts bank statements from users, detects subscriptions and recommends cancellation based on value.
StatementSense helps users save money by analyzing their spending history, device usage, the events they attend, and how much time they spend on their subscriptions
to recommend which ones to pause or cancel, suggest better alternatives, and identify plans that make more sense for the time of year.


# Subscription Features Evaluated
  - Usage Intensity   
  - Cost Efficiency   
  - Number of Charges 
  - Free Trial Probability
  - Monthly Charge (JMD)
  - User Priority
  - Duration of Months
